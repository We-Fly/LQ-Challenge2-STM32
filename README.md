# LQ-Challenge2-STM32

## 介绍

该仓库包含三个分支，分别为三款不同开发板的积分赛实现

- master(默认分支无代码)
- stm32f103rc
- wio-terminal <当前分支>
- null

由于未知原因我的stm32f103rct6坏了，因此本分支为移植到Wio-terminal开发板的版本

## 安装方法

该项目采用`PlatformIO`构建，请自行安装vscode插件

## 使用说明

安装好VSCode的`PlatformIO`插件以后，打开此工程文件夹

如果底部没有出现小按钮，重启VSCode

`PlatformIO`会自动下载安装库（需要代理）
